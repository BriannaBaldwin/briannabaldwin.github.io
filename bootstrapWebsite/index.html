About

This website is to keep track of all the great recipes I run into! Sometimes I want to get creative and make a fun desert, while others I just want to cook up a quick 30 minute meal. Since it's so hard to weed out all the recipes I don't like from the ones I do, I created this personal webpage to make it easy. It's also the perfect way to have all of my favorite recipes from different websites in one area to pick from.
